df -h
du -skh ./*
du -h --max-depth=1		 dir

디렉토리 삭제 rmdir rd
디렉토리 트리 tree
파일 삭제 rm del
파일 복사 cp copy
파일 이동 mv move
이름 변경 mv rename
change directory cd cd

화면 정리 clear cls

파일 내용 표시 cat type

쉘 종료, 도스창 종료 exit exit
시간 표시 date time
그대로 출력 echo echo
환경변수 표시 set, env set
경로 보기 echo $PATH path
버전 정보 uname -a ver
파일 찾기 find find
파일 안의 문자열 찾기 grep findstr
긴 화면 페이지 단위 출력 more more

chmod		 attrib

프로세스 종료 kill, killall tskill
