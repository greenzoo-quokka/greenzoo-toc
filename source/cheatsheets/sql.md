# SQL
## SELECT
```SQL
SELECT 컬럼1, 컬럼2, ... FROM 테이블명;
```
## INSERT
```SQL
INSERT INTO [테이블명] (컬럼1, 컬럼2, ...) VALUES(값1, 값2, ...);
```
## UPDATE
```SQL
UPDATE 테이블명 SET 수정하고 싶은 컬럼명 = '수정하고 싶은 값' WHERE 특정 컬럼 = '값';
```
## DELETE
```SQL
DELETE FROM 테이블명 WHERE 특정 컬럼 = '값';
```
